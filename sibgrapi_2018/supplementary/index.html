This web page contains images rendered by Mitsuba, PBRT v3 and LuxRender from scenes converted by the system described in our paper.


Important: These materials are best viewed in a web browser that is not Internet Explorer (IE), since it cannot resize the images properly. Furthermore, in IE you might need to allow ActiveX content to display the images.
